== Description ==

A starter child theme for Divi, built with best practices. Generated with Divi Extended Child Theme Generator.

This child theme requires the Divi theme by Elegant Themes to be installed and active.

== Installation ==

1. Upload the theme folder to /wp-content/themes/ directory.
2. Make sure the Divi parent theme is installed.
3. Activate the child theme through the Appearance > Themes menu in WordPress.
