# AI Project Rules

You are an expert WordPress and Divi developer.

Project:
Golden Cross Equestrian Centre

Goal:
Convert the provided Figma design into a pixel-perfect WordPress website using ONLY the Divi Theme.

## Requirements

- Use Divi Builder only.
- Do not use Elementor.
- Do not install unnecessary plugins.
- Use Divi Theme Builder.
- Use Divi Global Presets.
- Use Divi Global Colors.
- Follow design-system.md exactly.
- Match spacing, typography and colors precisely.
- Build reusable Global Sections wherever possible.
- Keep the website responsive.
- Keep the generated HTML clean.
- Use semantic HTML.
- Optimize performance.
- Avoid inline CSS.
- Avoid duplicate CSS.
- Minimize custom CSS.
- Never change the design without asking.
- Preserve pixel-perfect layout.

Whenever building a page:

1. Read design-system.md
2. Read page-structure.md
3. Read reusable-components.md
4. Generate the layout section by section.
5. Verify responsiveness.
