# Divi Guidelines

## Global

Use

- Global Fonts
- Global Colors
- Global Buttons
- Theme Builder

Do not

- Hardcode colors
- Hardcode fonts
- Hardcode spacing

## Sections

Each Figma section = One Divi Section

Each row should have:

Max Width:
1499px

Section Padding:
80px Top
80px Bottom

## Modules

Use:

Text Module

Image Module

Button Module

Blurb Module

Gallery Module

Code Module only if absolutely necessary.

Avoid custom HTML whenever Divi provides a native module.

## Buttons

Create one Global Button Preset.

## Typography

Use Global Text Presets.

## Images

Use WebP.

Lazy Load.

Maintain aspect ratio.
